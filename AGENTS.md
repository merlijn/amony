# Amony media server

This is a media server project. It has two main components:

- A backend written in Scala 3
- A frontend written in TypeScript with React

Each are in their own subdirectory. The backend provides an API that the frontend uses. 

It is generated by running `sbt generarateSpec` in the backend folder
It can be used by running `npm run generate` in the frontend folder, which generates the source code from the OpenAPI spec.
